# hpe_storage_flowkit_py
HPE Storage Flowkit for WSAPI v3

## Documentation

- [Volume Workflow](./volume_workflow.md): Details on creating and managing volumes using the workflow API, including parameter validation, payload construction, and error handling.
- [Volume Set Workflow](./volumeset_workflow.md): Comprehensive guide on managing volume sets (application sets), including create, delete, modify operations, and intelligent member management with add/remove capabilities.
- [Host Set Workflow](./hostset_workflow.md): Complete guide on managing host sets, including create, delete, get operations, member add/remove capabilities, and automatic handling of asynchronous task operations (202 responses).

Refer to the documentation for usage examples and API details.
